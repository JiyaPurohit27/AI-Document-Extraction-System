# AI-Powered Document Information Extraction System

## Project Objective
Develop a system that extracts key information from invoices and business documents using OCR and converts the extracted information into structured JSON format.

## Technologies Used
- Python
- EasyOCR
- OpenCV
- Pandas
- Streamlit
